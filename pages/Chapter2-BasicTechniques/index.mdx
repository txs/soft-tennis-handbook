# 第2章：基本動作與技巧

## 握拍方式

軟式網球中常見的握拍方式包括：

1. **東方式握拍法（Eastern Grip）**
   - 基本且常見，適合初學者
   - 適用於正手和反手擊球
   - 易產生上旋，適合多變球路

2. **半西方式握拍法（Semi-Western Grip）**
   - 適合中高級球員，主要用於正手
   - 易產生強烈上旋，適合高彈起的球

3. **大陸式握拍法（Continental Grip）**
   - 用於發球、網前截擊和切球
   - 便於快速轉換擊球角度

選擇握拍方式時，考慮個人風格、技術水平、身體條件和比賽策略。在教練指導下嘗試不同握法，找出最適合自己的方式。

## 基本站姿與步法

正確的站姿和靈活的步法是軟式網球的基礎。

### 準備姿勢

- 雙腳與肩同寬，重心略前傾
- 膝蓋微彎，上身稍前傾
- 雙臂自然彎曲，握拍手臂放鬆
- 目光集中在球和對手身上

### 基本步法

1. **側步**：短距離左右移動
2. **交叉步**：大範圍移動
3. **跑步**：長距離移動
4. **調整步**：微調身體位置
5. **回復步**：回到場地中心

### 練習建議

- 影子練習：專注腳步移動
- 敏捷梯訓練：提高靈活性
- 反應訓練：提高判斷和啟動速度
- 場地覆蓋練習：練習快速移動
